# Lumina LaTeX Editor — Stage 1G TeXlyre BusyTeX

Stage 1G keeps all Stage 1E/1F features and adds an additional browser-only compiler option:

- Backend TeX Live runner
- Mock draft validation
- SwiftLaTeX browser-WASM experimental provider
- **TeXlyre BusyTeX browser-WASM experimental provider**

Deploy the static app as before by uploading this folder to your GitHub Pages `Latexai` project. TeXlyre assets are not bundled because they are large; see `TEXLYRE_INSTALL_STAGE1G.md`.
